# Κεντρική ιδέα
Αρχικά θέλουμε η root process να φτιάξει l splitters μεσω fork() και να επικοινωνει με τον καθε ενα 
μεσω ενος ξεχωριστου pipe. Η root πρεπει να στειλει στο read end του καθε pipe το καταλληλο κομματι του 
input file που αντιστοιχει στο 1/l του συνολικου αριθμου γραμμων του αρχειου. Ετσι, κάθε splitter
αναλαμβάνει το δικό του κομμάτι αρχείου, το οποίο πρέπει να επεξεργαστεί κατάλληλα.
