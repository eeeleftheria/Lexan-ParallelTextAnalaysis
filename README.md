

##### DESTROY FUNCTIONS ######
Για την αποδεσμευση της μνημης για καθε γραφο που δημιουργειται πρεπει να καλεσουμε την GraphDestroy, η οποια ειναι υπευθυνη για την αποδεσμευση μνημης του graph και της λιστας nodes. Εκεινη με τη σειρα της θα καλεσει την listDestroy που διατρεχει τη λιστα μας και καταστρεφει τα value των κομβων μεσω της listDestoryValue. Στην συγκεκριμενη περιπτωση τα values των κομβων της βασικης λιστας με τις κορυφες (List nodes), αποτελουν δεικτες σε λιστες που συμπεριλαμβανουν τους γειτονες καθε κορυφης (List neighbors). Οποτε, για να γινει σωστα η αποδεσμευση πρεπει να διαγραφουν απο τη μνημη και η εσωτερικη λιστα με τους κομβους της.
